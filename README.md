# BRAND25 — Лендинг «Сайт за 24 часа за 30 000 ₽»

Полная структура и тексты для лендинга услуги BRAND25 в стиле современного минимализма.

## Структура проекта

```
brand25/
├── index.html              # Основная HTML-структура лендинга
├── styles.css              # Стили (светлая тема, минимализм)
├── script.js               # Базовая функциональность (форма, прокрутка)
├── design-guidelines.md    # Рекомендации по дизайну и стилю
├── ad-copy.md              # Рекламные тексты для соцсетей
└── README.md               # Этот файл
```

## Структура лендинга

1. **Hero-секция**
   - Главный заголовок с ценой
   - Подзаголовок с УТП
   - Две кнопки (основная и вторичная)
   - Три ключевых преимущества

2. **Как это работает**
   - 3 шага процесса
   - Простое объяснение без лишних деталей

3. **Для кого**
   - Личный бренд
   - Салоны красоты
   - Малый бизнес

4. **Что входит**
   - 6 пунктов включённых услуг
   - Чёткое описание каждого пункта

5. **Примеры работ**
   - 3 карточки с примерами сайтов
   - Скриншоты и краткое описание

6. **Отзывы**
   - 3 отзыва от клиентов
   - Имя, роль, текст отзыва

7. **Форма заявки**
   - Поля: имя, телефон, email, тип бизнеса, сообщение
   - Крупная кнопка отправки
   - Примечание о согласии на обработку данных

8. **Футер**
   - Логотип и слоган
   - Контакты
   - Копирайт

## Использование

### Для Figma
1. Откройте `index.html` в браузере для просмотра структуры
2. Используйте тексты из HTML для создания макетов
3. Следуйте рекомендациям из `design-guidelines.md`

### Для Tilda / Webflow
1. Скопируйте тексты из соответствующих секций `index.html`
2. Используйте структуру как основу для блоков
3. Примените стили из `styles.css` как референс

### Для разработки
1. Откройте `index.html` в браузере
2. Подключите `styles.css` и `script.js`
3. Настройте отправку формы на ваш бэкенд в `script.js`
