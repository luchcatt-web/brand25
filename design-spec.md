# BRAND25 — Техническое задание для дизайнера/разработчика

## Концепция
Ультра-компактный лендинг в стиле Apple — максимум 2 скролла, премиальное восприятие, glassmorphism эффекты.

---

## Структура (2 блока)

### БЛОК 1: HERO (первый экран, 100vh)

**Содержимое:**
- Заголовок: "Сайт за 24 часа для вашего бренда" (2 строки)
- Подзаголовок: "Готовый продающий сайт для личных брендов, салонов красоты и малого бизнеса. Фиксированная цена. Оплата после создания."
- Две стеклянные кнопки:
  - Основная: "Получить сайт за 24 часа"
  - Вторичная: "Примеры работ"
- Три фишки под кнопками (glassmorphism карточки):
  - ✓ Сайт за 24 часа
  - ✓ 30 000 ₽
  - ✓ Оплата после создания (выделить визуально)

**Визуал:**
- Фоновое изображение: красивое фото (мастер за работой, салон, локальный бизнес, работа за ноутбуком)
- Overlay: мягкий градиент (молочный, серо-голубой)
- Стиль: чисто, дорого, много воздуха

---

### БЛОК 2: WHY + BENEFITS + CTA (второй экран, 100vh)

**Содержимое:**
- Мини-заголовок: "Почему BRAND25?"
- 4 карточки с выгодами (grid 2x2):
  1. Индивидуальный дизайн — "Уникальный сайт, отражающий ваш бренд"
  2. Продающие тексты — "Копирайтинг, который конвертирует"
  3. Подключение заявок — "WhatsApp и Telegram интеграция"
  4. Готов к рекламе — "Через 24 часа можно запускать трафик"
- Финальная стеклянная кнопка: "Запустить сайт"
- Минималистичный футер

**Визуал:**
- Фон: молочно-белый (#FAFBFC)
- Карточки: glassmorphism с мягкими тенями
- Иконки: минималистичные SVG

## Цветовая палитра

### Основные цвета
- **Фон основной**: #F5F7FA (светло-серый)
- **Фон молочный**: #FAFBFC (почти белый)
- **Текст основной**: #1D1D1F (почти черный)
- **Текст вторичный**: #6E6E73 (серый)
- **Акцент**: #007AFF (iOS blue)
- **Акцент светлый**: #5AC8FA (голубой)

### Градиенты
- **Мягкий градиент**: linear-gradient(135deg, #F5F7FA 0%, #E8ECF1 100%)
- **Голубой градиент**: linear-gradient(135deg, #E8F4FD 0%, #F0F8FF 100%)

### Glassmorphism
- **Фон стекла**: rgba(255, 255, 255, 0.7)
- **Backdrop blur**: blur(20px) saturate(180%)
- **Граница**: rgba(255, 255, 255, 0.3)

## Типографика

### Шрифт
- **Основной**: SF Pro Display / -apple-system / System fonts
- **Размеры**:
  - Hero заголовок: 48–72px (clamp)
  - Подзаголовки: 19–24px
  - Основной текст: 17px
  - Мелкий текст: 13–15px

### Настройки
- Line-height: 1.47059 (Apple стандарт)
- Letter-spacing: -0.015em для заголовков
- Font-weight: 400 (regular), 500 (medium), 600 (semibold)

## Адаптивность

### Breakpoints
- Mobile: до 768px
- Tablet: 768px–1024px
- Desktop: от 1024px

### Правила
- На мобильных: одна колонка, кнопки на всю ширину
- Grid автоматически адаптируется
- Шрифты масштабируются через clamp()
- Отступы уменьшаются на мобильных

---

## Анимации

### Плавные переходы
- Все интерактивные элементы: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
- Hover эффекты: translateY(-2px до -8px)
- Модальное окно: slide in + scale

## Технические детали

### Backdrop-filter поддержка
- Использовать fallback для старых браузеров
- Проверка: @supports (backdrop-filter: blur())

### Производительность
- Оптимизировать изображения (WebP, lazy loading)
- Минимизировать количество DOM элементов
- Использовать CSS transforms для анимаций
